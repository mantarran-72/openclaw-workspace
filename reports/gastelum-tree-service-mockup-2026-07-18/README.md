# Gastelum Tree Service Mockup

Open `index-standalone.html` for the easiest preview. It embeds the Yelp photos directly, so it can be opened by itself.

`index.html` is the lighter editable version. Keep the `assets/` folder next to it so photos load.

## Source Notes

- Yelp Fusion API matched `Gastelum Tree Service`, alias `gastelum-tree-service-dixon-21`.
- Yelp API fields used: phone, rating, review count, categories, address, service locations, hours, business description, year established, photos, and review snippets.
- Official website checked: `https://gastelumtreeservice.com/`.
- Public search found the site has a basic call/contact flow but thin/repetitive content, so the lead status is `weak`, not `missing`.
- BuildZoom page found license and contractor details for `Gastelum Tree & Landscaping Services A Partnership`; verify against CSLB before publishing license/bond/insurance language on a final live site.

## Claims Used

- Established in 1991: sourced from Yelp API business attributes and their current website's 30+ years messaging.
- Tree services, landscaping, irrigation: sourced from Yelp categories.
- Emergency tree service, stump grinding, hazardous tree removal, mistletoe tree removal: sourced from Yelp API business attributes.
- Rating/review count/review snippets: sourced from Yelp API.
- Service areas: selected from Yelp API service location data plus owned-site regional language.

## Claims Avoided

- Certified Arborist
- ISA credentials
- Fully insured/bonded claims in customer-facing copy until verified directly
- Awards or guarantees
- Specific completed project claims beyond what the photos visibly support

## Sales Angle

This is not a "no website" prospect. It is a "weak owned website" prospect.

Respectful opener:

> I saw you already have a basic site, so I mocked up a cleaner quote-focused version using your real job photos, Yelp review highlights, and service categories. The idea is to make it easier for customers to understand what you do and call/send photos for an estimate.
