# Pacific Houseworks Proposal Template

Files in this folder:

- `construction-proposal-template.md` - editable text version for fast proposal drafting.
- `proposal-template-print.html` - branded print-ready version with logos.
- `assets/pacific-houseworks-monogram.jpg` - square PH logo.
- `assets/pacific-houseworks-wordmark.jpg` - Pacific Houseworks wordmark.

Suggested workflow:

1. Fill in `construction-proposal-template.md` first.
2. Copy the filled content into `proposal-template-print.html`, or edit the HTML directly if you want the branded layout.
3. Open `proposal-template-print.html` in a browser and print to PDF.
4. Keep detailed labor/material/subcontractor costs internal; the client-facing proposal should show base price, allowances, alternates, exclusions, payment schedule, and change-order terms.
